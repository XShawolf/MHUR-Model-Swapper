# Skin Swapper for My Hero Ultra Rumble
Tool to swap which skin slot a costume mod replaces in My Hero Ultra Rumble.

<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/5672f20d-5e0d-4c0f-aed0-fe78be602baf" />

# Usage
- Open SkinSwapper.exe
- If you're prompted to choose the games PAK file, look for it in the game's local files ``HerovsGame-WindowsNoEditor``
- Choose the mod file which you want to modify
- Choose the skin which you want the mod to go over
- Save the new mod file

# Dependencies
- UAssetAPI
- repak
- UE4-DDS Tools by matyalatte
- ffmpeg
